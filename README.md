# 20-js-projects-in-7-days-challenge

This repository contains 20 small JavaScript projects built over 7 days.
The aim of this challenge is to practice JavaScript fundamentals, DOM manipulation, and basic interactive web development.

Technologies Used ->

HTML5

CSS3

JavaScript (ES6+)

Fetch API (for API-based projects)

LocalStorage

How to Use: 

Open the project folder you want to run.

Open index.html in a browser.

Or use Live Server in VS Code for live preview.

Project Structure:
/project-folder/
│── index.html
│── style.css
└── script.js


Each project is self-contained and easy to run.

Challenge Details:

Duration: 7 days

Total Projects: 20

Focused on hands-on learning and practical implementation
